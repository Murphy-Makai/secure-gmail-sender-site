Secure Gmail Sender

A simple, secure tool that helps you send Gmail programmatically using OAuth 2.0.

About

Secure Gmail Sender is an application that uses Google OAuth 2.0 for secure access to Gmail accounts. The app never stores user passwords — only the OAuth tokens required to send email on behalf of consenting users. Tokens are stored securely and are refreshed using Google’s recommended flows.

How it works

  1. The user authorizes the app using the Google OAuth consent screen.
  2. The app receives an authorization code and exchanges it for an access token + refresh token.
  3. The app uses the access token to call Gmail API and may refresh tokens using the refresh token when needed.

Security & Data Handling

  • We only request the minimum Gmail scopes required to send messages.
  • Refresh tokens are stored encrypted (you must configure your storage back-end).
  • We never log plain-text tokens or user passwords.
