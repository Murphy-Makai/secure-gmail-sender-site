# Secure Gmail Sender — GitHub Pages Site

This repository hosts the public website required for **Google OAuth Verification** of the **Secure Gmail Sender** application.

It includes the following pages:

- **index.html** — Homepage
- **privacy.html** — Privacy Policy (Gmail API–compliant)
- **terms.html** — Terms of Service
- **style.css** — Global styling for all pages

## Purpose
Google requires apps using sensitive scopes (such as Gmail API) to provide publicly accessible pages detailing:

- What the app does
- How user data is used
- How tokens and information are secured
- Contact information

This GitHub Pages site fulfills those requirements.

## How to Publish via GitHub Pages
1. Go to **Settings → Pages**.
2. Under **Build and deployment**, select:
   - **Source:** Deploy from a branch
   - **Branch:** main
   - **Folder:** /** (root)
3. Save.

Your public website will appear at:
```
https://github.com/Murphy-Makai/secure-gmail-sender-site
```
